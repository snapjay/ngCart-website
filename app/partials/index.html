ngCart

Really simple shopping cart for AngularJS

Quick and easy AngularJS module and set of directives for adding a cart into your AngularJS application.
  • View cart directive
  • Add item to cart directive
  • Summary of items in your cart directive
  • Template driven for full UI customisation
  • Custom data object for additional info/options images
  • Add shipping costs
  • Add Taxes
  • Events Broadcast on $rootScope
  • Fulfilment options
    • Send to console
    • Send to HTTP POST
    • Send to PayPal
